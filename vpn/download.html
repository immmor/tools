Windows 安装教程

1

下载客户端

下载并安装 Clash for Windows 客户端

下载客户端
2

导入配置文件

打开 Clash for Windows，在 Profiles 页面：

  • • 点击 "Download from a URL"
  • • 粘贴配置链接：https://api.immmor.com/free/clash
  • • 点击 "Download" 导入配置
3

启动连接

在 General 页面：

  • • 选择刚才导入的配置文件
  • • 打开 "System Proxy" 开关
  • • 点击 "TUN Mode" 启用
  • • 选择节点并连接

Android 安装教程

1

安装客户端

从 Google Play 或 GitHub 下载 Clash for Android

Google Play GitHub
2

配置订阅

打开应用后：

  • • 点击右下角 "+" 按钮
  • • 选择 "从URL导入"
  • • 输入配置链接：https://api.immmor.com/free/clash
  • • 点击 "确定" 保存
3

启动VPN

返回主界面：

  • • 选择刚才添加的配置
  • • 点击右上角开关启动VPN
  • • 授予VPN连接权限
  • • 选择节点开始使用

iOS 安装教程

macOS 安装教程

1

下载客户端

下载 ClashX Pro 或 Clash for Windows macOS版

ClashX Pro CFW macOS
2

配置订阅

以 ClashX Pro 为例：

  • • 打开 ClashX Pro
  • • 点击菜单栏图标 → Config → Remote Config
  • • 点击 "Add" 添加订阅
  • • URL 输入：https://api.immmor.com/free/clash
3

启动服务

配置完成后：

  • • 点击菜单栏 ClashX 图标
  • • 选择 "Set as System Proxy"
  • • 选择 "Proxy" 模式
  • • 选择节点并测试连接

Linux 安装教程

1

安装 Clash

使用包管理器安装 clash：

# Ubuntu/Debian
sudo apt update && sudo apt install clash

# CentOS/RHEL
sudo yum install clash

# Arch Linux
sudo pacman -S clash
2

下载配置

下载配置文件到指定目录：

cd ~/.config/clash
wget https://api.immmor.com/free/clash -O config.yaml
3

启动服务

启动 clash 服务：

# 启动 clash
clash -d ~/.config/clash

# 设置系统代理（可选）
export http_proxy=http://127.0.0.1:7890
export https_proxy=http://127.0.0.1:7890
