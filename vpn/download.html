Windows 安装教程

1

下载客户端

下载并安装 FlClash 客户端

下载客户端
2

导入配置文件

打开 Clash for Windows，在 Profiles 页面：

  • • 点击 "Download from a URL"
  • • 粘贴配置链接：https://api.immmor.com/free/clash
  • • 点击 "Download" 导入配置
3

启动连接

在 General 页面：

  • • 选择刚才导入的配置文件
  • • 打开 "System Proxy" 开关
  • • 点击 "TUN Mode" 启用
  • • 选择节点并连接

Android 安装教程

1

安装客户端

从 GitHub 下载 FlClash

下载客户端
2

配置订阅

打开应用后：

  • • 点击右下角 "+" 按钮
  • • 选择 "从URL导入"
  • • 输入配置链接：https://api.immmor.com/free/clash
  • • 点击 "确定" 保存
3

启动VPN

返回主界面：

  • • 选择刚才添加的配置
  • • 点击右上角开关启动VPN
  • • 授予VPN连接权限
  • • 选择节点开始使用

iOS 安装教程

1

安装客户端

Shadowrocket
2

导入配置

以 Shadowrocket 为例：

  • • 点击右上角 "+" 按钮

连接使用

配置完成后：

  • • 在主界面选择订阅
  • • 点击开关启动连接
  • • 允许添加VPN配置
  • • 选择延迟最低的节点

macOS 安装教程

1

下载客户端

下载 FlClash macOS版

下载客户端
2

配置订阅

以 ClashX Pro 为例：

  • • 打开 ClashX Pro
  • • 点击菜单栏图标 → Config → Remote Config
  • • 点击 "Add" 添加订阅
  • • URL 输入：https://api.immmor.com/free/clash
3

启动服务

配置完成后：

  • • 点击菜单栏 ClashX 图标
  • • 选择 "Set as System Proxy"
  • • 选择 "Proxy" 模式
  • • 选择节点并测试连接
