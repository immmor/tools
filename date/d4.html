⚠️ 完整视频通话需要 Cloudflare Calls

      

        完整的点对点视频通话需要 SFU（Selective Forwarding Unit）服务。

        当前测试模式仅验证本地摄像头和 TURN credentials 是否可用。


        要启用完整功能：

        1. 在 Cloudflare Dashboard 中找到 Realtime → Calls

        2. 创建一个 Calls App

        3. 获取 App ID 并设置为环境变量 TURN_APP_ID

        4. 重新部署 Worker
