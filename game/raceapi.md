# 12选5赛车游戏 - 后端接口文档

**基础地址**: `http://localhost:8080`
**数据格式**: JSON (`Content-Type: application/json`)

---

## 1. 获取游戏状态 & 上一期开奖结果

```
GET /api/game/status
```

**请求参数**: 无

**响应示例**:
```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "issue_no": "No.2026001",
    "state": "Betting",
    "time_left": 32,
    "last_result": {
      "issue_no": "No.2026000",
      "top_5_cars": [7, 2, 11, 5, 9]
    }
  }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `issue_no` | string | 当前期号 |
| `state` | string | 当前状态：`Betting`(下注中) / `Drawing`(开奖中) |
| `time_left` | int | 当前阶段剩余秒数（倒计时） |
| `last_result` | object | 上一期开奖结果，首期为 `null` |

## 2. 玩家下注

```
POST /api/game/bet
```

**请求体**:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `order_id` | string | 是 | 订单唯一标识 |
| `play_mode` | string | 是 | 玩法类型（见下方枚举） |
| `selects` | int[] | 是 | 选中的赛车号码（1-12） |
| `amount` | float64 | 是 | 投注金额 |

### 玩法 (play_mode)

| 玩法 | 选号数 | 赔率 | 规则 |
|------|--------|------|------|
| `Top2` | 2 | 30x | 猜中前2名（顺序一致） |
| `Top3` | 3 | 110x | 猜中前3名（顺序一致） |
| `Top4` | 4 | 240x | 猜中前4名（顺序一致） |
| `Pick2` | 2 | 3x | 猜中前5名中任意2个 |
| `Pick3` | 3 | 10x | 猜中前5名中任意3个 |
| `Pick4` | 4 | 50x | 猜中前5名中任意4个 |
| `Pick5` | 5 | 400x | 猜中全部前5名 |
| `Pick6` | 6 | 60x | 前5名全在你的6个号里 |
| `Pick7` | 7 | 18x | 前5名全在你的7个号里 |
| `Pick8` | 8 | 7x | 前5名全在你的8个号里 |

**请求示例**:
```json
{
  "order_id": "bet_001",
  "play_mode": "Pick4",
  "selects": [3, 7, 11, 2],
  "amount": 100.00
}
```

**响应示例 (成功)**:
```json
{ "code": 200, "msg": "下注成功", "data": null }
```

**错误响应**:

| code | msg | 场景 |
|------|-----|------|
| 400 | 参数格式错误 | JSON 解析失败 |
| 403 | 当前不在下注时间段内 | 处于开奖阶段 |
| 400 | 选号数量不符合玩法 [...] 的要求 | selects 数量与 play_mode 不匹配 |

## 3. 取消下注（撤单）

```
POST /api/game/cancel
```

**请求体**:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `order_id` | string | 是 | 要取消的订单 ID |

**请求示例**:
```json
{ "order_id": "bet_001" }
```

**响应示例 (成功)**:
```json
{ "code": 200, "msg": "订单取消成功，金额已退回", "data": null }
```

**错误响应**:

| code | msg | 场景 |
|------|-----|------|
| 400 | 参数格式错误 | JSON 解析失败 |
| 403 | 已截止下注，无法取消订单 | 进入开奖阶段后锁死 |
| 404 | 订单不存在或已取消 | order_id 未找到或已删除 |

## 游戏流程时序

```
┌─────────────── 45s ───────────────┐   ┌──── 15s ────┐
│         Betting (下注阶段)          │ → │ Drawing(开奖) │
│  可调用 bet / cancel 接口           │   │  锁定，不可操作 │
└────────────────────────────────────┘   └──────────────┘
                    ↓ 循环
              下一期自动开始
```

> **注意**: 演示模式下周期为 **45秒下注 + 15秒开奖**，生产环境可调整为 **4分30秒 + 30秒**。
